# How to use
To execute interactive scripts use:
```bash
bash <(curl -sL https://raw.githubusercontent.com/matheusdgdcampos/scripts/main/PUT_FILENAME_HERE.sh)
```

To execute non interactive scripts use command bellow:
```bash
curl -o- https://raw.githubusercontent.com/matheusdgdcampos/scripts/main/PUT_FILENAME_HERE.sh | bash
```
